Documentación de IoT
====================

El objetivo de la documentación esta documentación se busca mostrar los pasos
a seguir para realizar una simulación de obtención de datos de las motas
zolertia z1 provistas por la cátedra.
La idea es poder montar una red con las motas mensionadas, obtener información
de sus sensores, enviar esa información fuera de la red de sensores,
almacenarla y visualizarla.
